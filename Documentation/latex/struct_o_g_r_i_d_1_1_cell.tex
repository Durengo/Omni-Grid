\doxysubsection{Detailed Description}
The \doxylink{struct_o_g_r_i_d_1_1_cell}{Cell} struct represents a single cell in the grid. 

It contains a pointer to a \doxylink{class_o_g_r_i_d_1_1_piece}{Piece} and the row and column of the cell. The \doxylink{class_o_g_r_i_d_1_1_piece}{Piece} pointer can be nullptr if the cell is empty. The row and column are unsigned chars, which means the grid can be at most 255x255. This is more than enough for our purposes. \begin{DoxyDate}{Date}
