\doxysubsection{Detailed Description}
The \doxylink{class_o_g_r_i_d_1_1_grid}{Grid} class represents a 2D grid of Cells. 

It contains the number of rows and columns in the grid, as well as the grid itself. The \doxylink{class_o_g_r_i_d_1_1_grid}{Grid} is a 2D array of \doxylink{struct_o_g_r_i_d_1_1_cell}{Cell} pointers. The \doxylink{class_o_g_r_i_d_1_1_grid}{Grid} can be at most 255x255, which is more than enough for our purposes. The \doxylink{class_o_g_r_i_d_1_1_grid}{Grid} class also contains a default \doxylink{class_o_g_r_i_d_1_1_piece}{Piece}, which is used to reset the grid. \begin{DoxyDate}{Date}
